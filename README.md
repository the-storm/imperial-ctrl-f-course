# CTRL+F: A Practical Guide to Security Reviews

**Course website:** https://ctrlf.cybersec.fun/

A 4-session mini-course at Imperial College London focused on code security auditing through hands-on labs.

## Getting Started

Clone the repo with all submodules:

```bash
git clone --recurse-submodules https://github.com/the-storm/imperial-mini-course-2025.git
```

If you already cloned without submodules, initialise them afterwards:

```bash
git submodule update --init --recursive
```

To update submodules to their latest tracked commits:

```bash
git submodule update --remote --recursive
```
