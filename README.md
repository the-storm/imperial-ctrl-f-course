# CTRL+F Course Materials

Course materials for the Imperial College CTRL+F practical security review course.

## Setup

Clone the repository with submodules:

```bash
git clone --recurse-submodules https://github.com/the-storm/imperial-ctrl-f-course.git```

If you already cloned without submodules:

```bash
git submodule update --init --recursive
```

Most labs use Docker. See each course and session README for lab-specific startup commands.
